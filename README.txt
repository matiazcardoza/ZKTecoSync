ZKTeco Sync - Sistema de Sincronización de Asistencias
Versión 1.1

DESCRIPCIÓN:
Aplicación para sincronizar registros de asistencia desde dispositivos ZKTeco hacia el sistema central.

REQUISITOS:
- Windows 7 o superior
- Conexión de red
- Permisos de administrador para la instalación

INSTALACIÓN:
1. Ejecutar el instalador como administrador
2. Seguir las instrucciones en pantalla
3. La aplicación se instalará en Archivos de programa

USO:
1. Configurar la IP del dispositivo ZKTeco
2. Probar la conexión
3. Conectar al dispositivo
4. Extraer y sincronizar asistencias

SOPORTE:
Para soporte técnico, contactar al administrador del sistema.
